Для чего используются и чем отличаются атрибуты async и defer?

async - cкрипт выполнится, как только будет загружен, defer - скрипт будет выполнен после полной загрузки страницы

Страница будет быстрее загражаться и загружать скрипты.

Какие основные отличия интерпретируемого языка программирования от компилируемого?

Компилируемый - компилируется и выполняется сразу вся. Интерпретируемый - выполняется по одной строчке.

Что такое языки высокого и низкого уровня? Какие ты знаешь?

Низкоуровневый - близок к машинному коду. Высокоуровневый - разработан для быстроты и удобства использования программистом(JS, Java, C++, C, PHP).

Для чего используются переменные? Какие ключевые слова используются для создания переменных? Какие есть отличия в переопределении значений при задании переменной через let, const и var?

Переменные нужны для того, что-бы записывать туда значения для дальнейшего использования. Var(старое, её можно переопределить), let(нельзя переопределять на прямую), const(вообще нельзя переопределять).

Может ли имя переменной начинаться с цифры, символов _ или $?

C цифры не может начинаться, а с символов может.

Какие слова запрещено использовать в названии переменной?

Русские слова, называть переменные нужно так, что-бы назавание соответствовало хранимым в ней данных, цифры, нельзя использовать пробелы.

Что такое camelCase и snake_case?

camelCase - стиль написания где все пишется слитно, snake_case - стиль написания где место пробела "_".

Что такое "типы данных"? Какой оператор позволяет определить тип данных?

Типы данных - класс данных.typeof позволяет узнать тип данных.

Что такое "динамическое приведение типов данных"? Можно ли писать JavaScript со статическим приведением типов?

Динамическое приведение типов данных - механизм, который позволяет определить тип данных во время выполнения программы. Можна

Чем отличаются операторы "==" и "==="?

Это опереторы сравнения, "==" - равенство(приводит значения к одному типу), "===" - строгое сравнение.
